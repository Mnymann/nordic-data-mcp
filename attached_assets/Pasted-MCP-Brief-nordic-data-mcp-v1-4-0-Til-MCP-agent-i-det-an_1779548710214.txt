Endpoint-kontrakt
Method: GET
Path: /api/company/fr/:siren/history
Header: X-Api-Key: <tenant key> (samme som de øvrige tools)
Path param: siren — 9 cifre, fx 652014051 (Carrefour) eller 775670417 (LVMH)
Cost: 1 unit (FR er ikke i FREE_TIER_BLOCKED_COUNTRIES)
Cache: 24 h server-side; klient kan se X-Cache: HIT|MISS header

Mulige event types
initial:name, initial:usage_name, initial:activity, initial:status, initial:legal_form, initial:employer_flag, initial:ess_flag — én "initial:" per ikke-null felt i første periode (baseline)
name — denominationUniteLegale ændret
usage_name — denominationUsuelle1UniteLegale ændret
activity — NAF-kode (activitePrincipaleUniteLegale) ændret
status — A (active) ↔ C (cessée)
legal_form — categorieJuridiqueUniteLegale ændret (numerisk INSEE-kode)
employer_flag — caractereEmployeurUniteLegale ændret (O/N)
ess_flag — economieSocialeSolidaireUniteLegale ændret (O/N)
NDJSON streaming (valgfri)
Send Accept: application/x-ndjson for line-delimited output:

{"type":"meta","siren":"652014051","dateCreation":"1963-01-01","periodCount":9,"count":11,"source":"api.insee.fr","fetchedAt":"..."}
{"type":"event","eventType":"initial:name","date":"1963-01-01","endDate":"2004-12-24","from":null,"to":"CARREFOUR"}
{"type":"event","eventType":"activity","date":"2010-12-13","endDate":"2018-12-31","from":"70.10Z","to":"64.20Z"}

type = line discriminator (meta eller event). eventType = selve felt-typen.

Fejl
HTTP	Body	Hvornår
400	{ "error": "invalid_id_format", "expected": "9-digit SIREN number..." }	Ikke 9 cifre
401	(auth-middleware)	Manglende/forkert X-Api-Key
402	{ "error": "upgrade_required", ... }	Ikke relevant for FR — kun NL+DE er blokeret på free tier
403	{ "error": "non_diffusible", "siren": ..., "country": "fr", "message": "..." }	SIREN findes men er privatlivs-beskyttet (art. R123-232-1)
404	{ "error": "not_found", "siren": ..., "country": "fr" }	Ukendt SIREN
503	{ "error": "upstream_unavailable", "source": "api.insee.fr", "reason": "..." }	INSEE nede eller rate-limited
Tool description (forslag til Claude)
Name: fr_history
Description: French company history timeline. Returns one event per change
to the company's name, activity (NAF code), status (active/closed), legal
form, or social-economy flag, derived from INSEE Sirene 3.11's bitemporal
periodesUniteLegale array. Includes "initial:<field>" events showing the
state at company creation. Input: 9-digit SIREN number. Free tier: yes
(France costs 1 unit, not blocked).
Input schema: { siren: string (9 digits, e.g. "652014051" for Carrefour) }
