Her er den endelige, ferdige Prompt 2 — opdateret med alt vi lærte undervejs (kanonisk URL, gatet spec, scoped nøgle allerede i env). Klar til at fyre af mod MCP-agenten.

**Før du fyrer:** bekræft at MCP'ens base-URL er sat til `https://api.addonnordic.dk` — ikke den rå `.up.railway.app`-URL. Det var hele 401-årsagen, og spec-hentningen nedenfor afhænger af det.

behold de 8 eksisterende kuraterede tools UÆNDRET, og tilføj 3

https://api.addonnordic.dk/openapi.json

## Når den melder grønt

Du er i mål med det oprindelige formål: Claude Code kan nu opdage og teste alle ~250 endpoints selv — ingen copy-paste fra Replit igen.

Tre verifikationspunkter er de afgørende:
- **Punkt 1** (~250, ikke 8) = discovery virker.
- **Punkt 2–3** (admin afvist) = sikkerhedshærdningen holder.
- **Punkt 6** (8 gamle tools intakte) = ingen regression.

Derefter er kun **MCP-reviewet** tilbage — og takket være at sikkerheden er bygget ind her, bliver det en kort bekræftelse, ikke en ny runde.

Sig til når den er kørt, så kigger vi resultatet igennem sammen — og så er hele kæden backend → MCP → Claude Code lukket.
